# Generative AI API Integration

# Project Description
This project demonstrates integration of multiple Generative AI APIs using Python. Each program takes user input and generates AI-based responses.

# APIs Used
- Groq 
- Hugging Face
- Google Gemini
- Cohere
- Ollama (Tiny Llama)

# Setup Instructions
1. Install Python
2. Install required libraries:
   pip install -r requirements.txt

# How to Obtain API Keys

# Groq
- https://console.groq.com/
- Generate API key

# Hugging Face
- https://huggingface.co/settings/tokens
- Create token

# Gemini
- https://aistudio.google.com/
- Create API key

# Cohere
- https://dashboard.cohere.com/
- Generate API key

# Ollama
- Download from https://ollama.com/
- No API key required

# How to Run

Run each file:

python groq_example.py  
python huggingface_example.py  
python gemini_example.py  
python cohere_example.py  
python ollama_example.py
